# Padrao de Code Review

## Linguagens

- Frontend: Vue 3 + TypeScript
- Backend: PHP 8.3

## Checklist de revisao

1. Seguranca:
   - SQL Injection, XSS, CSRF, auth e autorizacao.
   - Segredos hardcoded.
2. Arquitetura:
   - Frontend separado de backend.
   - Responsabilidade unica por modulo.
3. Qualidade:
   - Legibilidade, duplicacao, nomes e complexidade.
4. Performance:
   - Consultas, loops, renderizacao e chamadas de API.
5. Infra:
   - Compatibilidade com Docker/Coolify.

## Formato esperado da revisao

1. Criticos
2. Avisos
3. Melhorias sugeridas
4. Proximos passos priorizados
